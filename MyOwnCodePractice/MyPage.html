  1. What is Chart Sorting Visualization?
    This visualization is to help visualize how each sorting algorithm work in the backend of things.
    Encapsulation prevents us from seeing how results are done before returning it back to us.
    Example of sorting algorithm:

    • Bubble Sort
    • Merge Sort
    • Select Sort
    • Insertion Sort

  2. What is Pathfinding Visualization?
    This visualization is to help visualize how a searching path algorithm work in the backend of things.
    Encapsulating prevents us from seeing how results are done before returning it to us.
    Example of path search algorithm

    • Dijkstra's Algorithm
    • A* algorithm
    • Sample Algorithm
